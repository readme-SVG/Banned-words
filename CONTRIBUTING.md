Thanks for contributing to this repository.

This project is a collection of banned words, so please follow these rules:

1. Do not add duplicate words.
2. Use one word per line.
3. Write each new word on a new line.
4. Every word must start with a lowercase letter.

Before committing, review the edited language file and make sure the format is still correct.
